Название проекта: Шахматное соревнование Это консольное приложение поможет вам быстро отсортировать данные участников шахматного турнира с помощью шейкерной сортировки.

Ключевые особенности:
Управление данными: Хранение полной информации об участниках, включая ФИО, дату рождения (раздельно по году, месяцу, числу), разряд, количество побед и страну.
Хранение: Использование простого текстового файла (chess_database.txt) для надёжного и доступного хранения данных. Реализована функция инициализации базы при первом запуске.
Алгоритм сортировки: Реализована классическая Шейкер-сортировка (Shaker Sort) для упорядочивания данных.
Фильтрация и Отчёты: Формирование трёх типов отчётов с возможностью фильтрации по стране и заданному диапазону разрядов.
